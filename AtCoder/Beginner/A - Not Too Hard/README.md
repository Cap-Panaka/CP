There is a programming contest with  N  problems. For each  i = 1, 2, …, N , the score for the  i -th problem is  S_i .

Print the total score for all problems with a score of  X  or less.

### Constraints

- All input values are integers.

- 4 ≤ N ≤ 8

- 100 ≤ S_i ≤ 675

- 100 ≤ X ≤ 675

## Input

The input is given from Standard Input in the following format:

```
N X
S_1 S_2 \ldots S_N
```

## Output

Print the answer.

## Examples

**Sample Input 1:**

```
6 200
100 675 201 200 199 328
```

**Sample Output 1:**

```
499
```

**Sample Input 2:**

```
8 675
675 675 675 675 675 675 675 675
```

**Sample Output 2:**

```
5400
```

**Sample Input 3:**

```
8 674
675 675 675 675 675 675 675 675
```

**Sample Output 3:**

```
0
```
